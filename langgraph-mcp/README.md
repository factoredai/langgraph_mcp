# LangGraph Simple Agent

## Setup

1. Copy [.env.example](.env.example) to `.env` and fill in the values.
2. Install uv

    ```bash
    curl -LsSf https://astral.sh/uv/install.sh | sh
    ```
3. Install dependencies

    ```bash
    uv sync
    ```


## Run in LangGraph Studio

```bash
langgraph dev
```
